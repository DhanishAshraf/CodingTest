# CodingTest
A program that parses a csv file containing information about routers and returns a 
list of the ones that need updating according to some criteria.

To run the program simply compile the class RouterPatchCheck.java in the command line and then run 
with a single command line argument stating the name of the .csv file with the extension.

e.g. javac RouterPatchChecker.java
     java RouterPatchChecker sample.csv
    
    
RouterPatchCheckerTest is the JUnit tests that were completed to esnure the program works properly.
